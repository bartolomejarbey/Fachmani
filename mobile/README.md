# Fachmani — iOS aplikace (Capacitor)

Nativní iOS obal kolem webové aplikace Fachmani (fachmani.org), připravený k odeslání do App Store.
Postaveno na Capacitoru (WKWebView shell + nativní pluginy). Web aplikace zůstává nedotčená.

## Struktura
- `docs/` — výzkum App Store pravidel + compliance checklist + runbook k odeslání
- `assets/` — ikony, splash, App Store metadata
- (po scaffoldu) `ios/` — Xcode projekt, `capacitor.config.ts`, `package.json`

Stav: rozpracováno (ultracode build). Viz docs/.
